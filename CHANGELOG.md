# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-04-15

### Added
- Initial release
- PSD file parsing and layer extraction
- JSON layout data export with precise positioning
- Automatic layer image export
- Thumbnail generation for AI-optimized reading
- Text layer content and style extraction
- Preview image generation
- Auto-generated README with AI prompt templates
- CLI tool support (`psd2ai` command)
- NPM package support
- Support for `npx psd2ai` usage

### Features
- Extract layer position, size, bounds, opacity
- Export individual layer images as PNG
- Generate optimized thumbnails (max 400px width)
- Extract text content, font size, and color
- Support for layer groups and nested structures
- Comprehensive documentation generation
- Zero-configuration usage
